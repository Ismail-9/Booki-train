# Booki - Projet 2 OpenClassrooms

Version modernisée du projet Réservia 2021 vers une version Booki actuelle.

## Contenu

- `index.html` : structure HTML sémantique.
- `styles.css` : mise en page desktop, tablette et mobile.

## Ce qui a été gardé de l'ancien Réservia

- La structure globale : header, recherche, filtres, hébergements, populaires, activités, footer.
- Le thème Marseille.
- Les cartes hébergements / populaires / activités.

## Ce qui a été modernisé

- Nom Booki au lieu de Réservia.
- Suppression du lien “S'inscrire”.
- Filtre “Nos pépites” à la place de “Animaux autorisés”.
- HTML plus sémantique avec `header`, `main`, `section`, `article`, `aside`, `footer`.
- CSS plus propre, responsive et maintenable.
- Corrections orthographiques : “Hébergements”, “Canebière”, “Vieux-Port”, etc.

## Responsive

- Desktop : hébergements et populaires côte à côte.
- Tablette : populaires en haut sur une ligne, hébergements ensuite.
- Mobile : populaires avant hébergements, cartes en une colonne, recherche adaptée avec icône loupe.
